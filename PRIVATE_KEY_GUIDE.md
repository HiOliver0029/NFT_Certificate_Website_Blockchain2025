# 🔑 獲取 MetaMask 私鑰詳細步驟

## ⚠️ 重要警告
- **僅用於測試網**，不要使用包含真實資金的錢包！
- 建議為測試專門創建一個新錢包

## 📱 獲取私鑰步驟

### 方法一: 從現有錢包獲取（推薦為測試創建新錢包）

1. **打開 MetaMask**
2. **點擊右上角的圓形頭像**
3. **選擇 "設定" (Settings)**
4. **選擇 "安全與隱私" (Security & Privacy)**
5. **點擊 "顯示私鑰" (Reveal Private Key)**
6. **輸入 MetaMask 密碼**
7. **複製完整的私鑰**（應該是 66 字符，以 0x 開頭）

### 方法二: 創建測試專用錢包（更安全）

1. **創建新帳戶**
   - MetaMask → 點擊帳戶圓圈 → "創建帳戶"
   - 命名為 "Sepolia Test Account"

2. **獲取測試ETH**
   - 複製新錢包地址
   - 前往 https://sepoliafaucet.com/
   - 貼上地址，獲取測試ETH

3. **導出私鑰**
   - 按照方法一的步驟導出私鑰

## 🧪 驗證私鑰格式

正確的私鑰應該看起來像這樣：
```
0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef
```

**檢查清單**:
- [ ] 以 `0x` 開頭
- [ ] 總長度 66 字符
- [ ] 只包含數字 0-9 和字母 a-f

## 🔒 安全提醒

1. **僅用於測試網** - 不要使用主網錢包
2. **不要分享私鑰** - 私鑰等於完全控制權
3. **使用專用測試錢包** - 與日常使用的錢包分開

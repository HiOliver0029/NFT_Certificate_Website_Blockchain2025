# 🦊 MetaMask 快速設置卡 - 3 分鐘搞定！

## ⚡ 快速設置（已有 MetaMask）

### 步驟 1: 切換到 Sepolia 測試網 (30秒)
```
1. 打開 MetaMask 🦊
2. 點擊頂部的網路名稱（顯示「Ethereum Mainnet」）
3. 選擇「Sepolia 測試網路」
   
   ⚠️ 如果看不到 Sepolia:
   - 點擊「顯示/隱藏測試網路」
   - 開啟「顯示測試網路」開關
```

### 步驟 2: 獲取測試 ETH (5-10分鐘)
```
1. 複製您的錢包地址
   - 點擊 MetaMask 中的地址（會自動複製）
   
2. 訪問 Faucet
   - https://sepolia-faucet.pk910.de/
   
3. 貼上地址並請求 ETH
   - 完成簡單的工作量證明
   - 等待 5-10 分鐘
```

### 步驟 3: 連接前端應用 (1分鐘)
```
1. 打開前端
   - http://localhost:3000 或 http://localhost:3001
   
2. 強制刷新瀏覽器
   - 按 Ctrl + Shift + R
   
3. 連接 MetaMask
   - 點擊「連接 MetaMask」按鈕
   - 在彈出視窗點擊「連接」
   
4. 查看您的證書！🎉
```

## 🆕 從零開始（沒有 MetaMask）

### 第一步: 安裝 MetaMask (2分鐘)
```
1. 訪問 https://metamask.io/
2. 點擊「Download」
3. 選擇您的瀏覽器並安裝
4. 點擊瀏覽器右上角的 MetaMask 圖標 🦊
```

### 第二步: 創建錢包 (3分鐘)
```
1. 點擊「開始使用」
2. 選擇「創建新錢包」
3. 設置密碼（至少 8 個字符）
4. 備份助記詞（12 個單詞）
   
   ⚠️ 重要: 
   - 抄寫在紙上
   - 絕對不要分享給任何人
   - 這是恢復錢包的唯一方式
   
5. 確認助記詞順序
6. 完成！
```

### 第三步: 按照「快速設置」操作
繼續執行上面的步驟 1-3

## 🚨 常見錯誤快速修復

### ❌ 「請安裝 MetaMask」
```
解決: 
1. 安裝 MetaMask 擴展
2. 刷新頁面
```

### ❌ 「請切換到 Sepolia 網路」
```
解決:
1. 打開 MetaMask
2. 點擊頂部網路名稱
3. 選擇「Sepolia 測試網路」
```

### ❌ 「載入證書失敗」
```
解決:
1. 按 Ctrl + Shift + R 強制刷新
2. 清除瀏覽器緩存 (Ctrl + Shift + Delete)
3. 重新連接 MetaMask
```

### ❌ 「餘額不足」
```
解決:
1. 確認在 Sepolia 測試網（不是主網！）
2. 從 Faucet 獲取測試 ETH
3. 等待幾分鐘
```

### ❌ MetaMask 沒有彈出
```
解決:
1. 檢查瀏覽器右上角是否有通知
2. 手動點擊 MetaMask 圖標 🦊
3. 確認錢包已解鎖（輸入密碼）
```

# 🎤 永恆數位榮譽證書 - 項目展示簡報

這是一個用於課堂展示的完整簡報套件，包含多種格式：互動式 HTML、PowerPoint (PPTX) 和 PDF，共 14-15 張投影片，全面介紹您的 NFT 證書項目。

## 📁 檔案結構

```
presentation/
├── index.html                          # 互動式 HTML 簡報
├── styles.css                          # HTML 簡報樣式表
├── script.js                           # 互動功能腳本
├── generate_ppt.py                     # PowerPoint 生成腳本
├── generate_pdf.py                     # PDF 生成腳本
├── 永恆數位榮譽證書_專案簡報.pptx      # PowerPoint 簡報 (可編輯)
├── 永恆數位榮譽證書_專案簡報.pdf       # PDF 簡報 (便攜式)
└── README.md                           # 本說明文件
```

## 🎯 三種簡報格式比較

| 格式 | 優點 | 適用場景 | 檔案大小 |
|------|------|---------|---------|
| **HTML** | 互動式、動畫效果、鍵盤控制、響應式 | 電腦演示、線上展示 | ~50KB |
| **PPTX** | 可編輯、Microsoft Office 原生、支援備註 | 正式簡報、需要修改內容 | ~100KB |
| **PDF** | 跨平台、無需特殊軟體、保真度高 | 列印、分享、歸檔 | ~150KB |

## 🚀 使用方法

### 選項 1: PowerPoint 簡報 (推薦用於正式場合) ⭐

**檔案**: `永恆數位榮譽證書_專案簡報.pptx`

1. **開啟方式**:
   - 雙擊 `.pptx` 檔案
   - 使用 Microsoft PowerPoint 開啟
   - 或使用 Google Slides（上傳到 Google Drive）
   - 或使用 LibreOffice Impress（免費）

2. **優點**:
   - ✅ 可以直接編輯內容
   - ✅ 支援備註功能（演講者筆記）
   - ✅ 與投影機/投影設備相容性最佳
   - ✅ 支援簡報者模式
   - ✅ 可以輕鬆添加動畫效果

3. **演示控制**:
   - **滑鼠點擊** 或 **空白鍵**：下一張
   - **Backspace**：上一張
   - **F5**：從頭播放
   - **Shift+F5**：從目前投影片播放
   - **Esc**：結束播放

### 選項 2: PDF 簡報 (推薦用於分享/列印)

**檔案**: `永恆數位榮譽證書_專案簡報.pdf`

1. **開啟方式**:
   - 雙擊 `.pdf` 檔案
   - 使用任何 PDF 閱讀器（Adobe Reader、瀏覽器等）

2. **優點**:
   - ✅ 跨平台通用（Windows、Mac、Linux）
   - ✅ 不需要特殊軟體
   - ✅ 適合列印成紙本
   - ✅ 檔案大小固定，易於分享
   - ✅ 版面保真度高

3. **演示控制**:
   - **方向鍵** ↑↓：翻頁
   - **F11**：全螢幕模式（大多數閱讀器）
   - **PageUp/PageDown**：翻頁

### 選項 3: 互動式 HTML 簡報 (推薦用於技術展示)

**檔案**: `index.html`

### 方法 1: 直接開啟（推薦）
1. 雙擊 `index.html` 檔案
2. 瀏覽器會自動開啟簡報
3. 使用以下控制鍵操作：
   - **箭頭鍵** ←→ 或 ↑↓：切換上一張/下一張投影片
   - **空白鍵**：下一張投影片
   - **Home**：跳到第一張
   - **End**：跳到最後一張
   - **F11**：全螢幕模式
   - **雙擊**：切換全螢幕

### 方法 2: 本地伺服器（避免 CORS 問題）
```bash
# 在 presentation 目錄下執行
python -m http.server 8000
# 或使用 Node.js
npx serve
```
然後在瀏覽器開啟 `http://localhost:8000`

### 方法 3: VS Code Live Server
1. 在 VS Code 中右鍵點擊 `index.html`
2. 選擇 "Open with Live Server"

**HTML 簡報優點**:
- ✅ 現代化的動畫效果
- ✅ 響應式設計（自動適應螢幕）
- ✅ 完整鍵盤控制
- ✅ 觸控滑動支援
- ✅ 無需安裝任何軟體

## 🎯 快速選擇指南

**我應該使用哪個格式？**

| 情境 | 建議格式 | 原因 |
|------|---------|------|
| 🎓 **課堂正式簡報** | PPTX | 投影機相容性佳，可用簡報者模式 |
| 📧 **Email 分享** | PDF | 檔案小、相容性好、不會跑版 |
| 💻 **技術展示/Demo** | HTML | 互動性強、視覺效果佳 |
| 🖨️ **列印講義** | PDF | 版面固定、適合列印 |
| ✏️ **需要修改內容** | PPTX | 可編輯性最佳 |
| 📱 **手機/平板查看** | HTML 或 PDF | 響應式或通用格式 |

## 🔄 重新生成簡報

如果您修改了內容想重新生成 PPTX 或 PDF：

```bash
# 重新生成 PowerPoint
cd presentation
python generate_ppt.py

# 重新生成 PDF
python generate_pdf.py
```

**注意**: 需要先安裝 Python 套件：
```bash
pip install python-pptx reportlab
```

## 🐛 疑難排解

### 問題：投影片無法切換
**解決**: 檢查 JavaScript 控制台是否有錯誤，確保 `script.js` 正確載入

### 問題：樣式跑版
**解決**: 確保 `styles.css` 檔案在同一目錄下，並檢查瀏覽器快取

### 問題：全螢幕無法啟動
**解決**: 某些瀏覽器需要使用者手勢觸發，嘗試 F11 鍵
